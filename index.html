LATAR BELAKANG

Pada era digital yang semakin berkembang, kemampuan untuk membuat dan memahami struktur dasar sebuah website menjadi semakin penting. Website tidak hanya menjadi alat komunikasi, tetapi juga sarana untuk berbagi informasi, produk, dan layanan kepada khalayak yang lebih luas. Dalam dunia pemrograman web, HTML (HyperText Markup Language) tetap menjadi salah satu bahasa yang paling fundamental dan penting. Pada saat yang sama, kemajuan dalam alat-alat pengembangan web telah mempermudah proses pembuatan website. Salah satu alat yang banyak digunakan oleh para pengembang web adalah Visual Studio Code (VSCode). VSCode adalah editor kode sumber yang ringan, kuat, dan fleksibel yang dikembangkan oleh Microsoft. Dengan antarmuka yang bersih dan ekosistem ekstensi yang kaya, VSCode menjadi pilihan populer bagi para pengembang web dari berbagai tingkatan keahlian

VISUAL STUDIO CODE

Visual Studio Code (VS Code) adalah sebuah editor kode sumber yang dikembangkan oleh Microsoft. Ini adalah perangkat lunak sumber terbuka dan tersedia secara gratis yang dapat dijalankan di berbagai platform, termasuk Windows, macOS, dan Linux. VS Code menawarkan berbagai fitur yang berguna bagi pengembang. Alat ini biasanya digunakan untuk pengembangan web, tetapi mendukung banyak bahasa pemrograman dan memiliki berbagai extension yang tersedia.

DEFINISI WEBSITE

Situs web adalah kumpulan halaman web dan konten terkait yang diidentifikasi dengan nama domain umum dan dipublikasikan di setidaknya satu server web. Ini dapat berisi teks, gambar, video, dan aset digital lainnya, yang dapat diakses melalui internet.

Kekurangan HTML

  1. HTML biasa digunakan untuk membuat halaman web statis sehingga jika ingin fitur yang dinamis perlu menggunakan bahasa backend, seperti PHP ataupun JavaScript.
  2. Tidak memungkinkan user atau pengguna untuk menjalankan logic.
  3. Fitur baru tidak dapat dijalankan secara cepat di sebagian browser.

Kelebihan HTML

  1. Bersifat open-source serta sepenuhnya gratis.
  2. Memiliki Bahasa Markup yang rapi dan konsisten.
  3. Memiliki banyak sumber dan komunitas yang besar.
  4. Mudah diintegrasikan dengan bahasa backend Dijalankan di setiap web browser.

Manfaat Website

Selain sebagai branding dan profil perusahaan, website juga bermanfaat untuk sumber informasi dan memudahkan komunikasi. Dengan memiliki website, Anda bisa dengan mudah memperbarui informasi bagi pengguna situs Anda. Pembaruan informasi juga dapat dilakukan secara cepat dan rutin.

BERIKUT LANGKAH LANGKAH MEMBUAT WEBSITE HTML DENGAN VSCODE

gambaran jika html terisi penuh

Template HTML adalah kerangka dasar atau struktur HTML yang digunakan sebagai dasar untuk membuat halaman web yang memiliki desain atau tata letak yang serupa. Template ini membantu mempercepat proses pengembangan web dengan menyediakan elemen-elemen dasar yang dapat diisi atau dimodifikasi sesuai kebutuhan. Template HTML biasanya mencakup bagian-bagian seperti header, footer, sidebar, dan area konten utama, yang bisa diisi dengan konten dinamis atau statis

Heading (atau heading) dalam HTML adalah elemen yang digunakan untuk menentukan judul atau subjudul pada sebuah halaman web. Heading membantu dalam membentuk struktur konten dan memberikan hierarki yang jelas, sehingga memudahkan pembaca dan mesin pencari untuk memahami konten yang ada. HTML menyediakan enam tingkat heading, dari 1 hingga 6, di mana 1 merupakan heading dengan tingkat tertinggi (biasanya digunakan untuk judul utama) dan 6 merupakan heading dengan tingkat terendah

Paragraf dalam HTML adalah elemen yang digunakan untuk mendefinisikan blok teks yang terpisah dan mandiri dalam sebuah halaman web. Elemen ini dilambangkan dengan tag p, yang merupakan singkatan dari "paragraph". Paragraf digunakan untuk menyusun teks agar mudah dibaca dan diatur secara logis, sehingga memudahkan pengunjung situs web untuk memahami konten.

Dalam HTML, daftar (list) digunakan untuk menyusun item-item menjadi bentuk yang terstruktur, baik itu daftar berpoin, bernomor, maupun daftar definisi.

Anchor HTML adalah elemen dalam HTML yang digunakan untuk membuat tautan atau hyperlink ke halaman web lain, bagian dari halaman yang sama, atau ke berkas lainnya. Ini ditandai dengan menggunakan elemen a dan atribut href untuk menentukan URL tujuan.

Elemen adalah elemen HTML yang digunakan untuk menampilkan gambar di halaman web. Elemen img tidak memiliki tag penutup (self-closing tag) dan memiliki atribut src yang digunakan untuk menentukan lokasi file gambar yang akan ditampilkan. Beberapa atribut lainnya yang sering digunakan adalah alt, yang memberikan teks alternatif untuk gambar untuk aksesibilitas dan SEO, serta width dan height untuk menentukan ukuran gambar dalam piksel. Contoh penggunaan elemen img

Elemen table dalam HTML digunakan untuk membuat tabel di dalam halaman web. Tabel terdiri dari baris dan kolom yang terorganisir secara terstruktur. Di dalam tabel, Anda dapat menempatkan teks, gambar, atau elemen HTML lainnya.

Elemen form dalam HTML digunakan untuk membuat formulir di dalam halaman web. Formulir memungkinkan pengguna untuk mengirimkan data ke server untuk diproses. Formulir biasanya terdiri dari elemen-elemen seperti input teks, tombol, kotak centang, dan sebagainya

Elemen iframe dalam HTML digunakan untuk menanamkan sebuah halaman web atau konten dari sebuah situs web ke dalam halaman web lain. Ini memungkinkan pengembang untuk menampilkan konten dari sumber eksternal di dalam halaman web mereka
